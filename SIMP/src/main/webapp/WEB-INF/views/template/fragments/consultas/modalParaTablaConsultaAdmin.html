Asocia una tarjeta a un cliente existente. El cliente no debe tener ninguna tarjeta asociada activa.
Aviso. La tarjeta ya esta asociada a un cliente
Formulario
Realiza la activación de la tarjeta. La tarjeta debe estar asociada a un cliente y encontrarse en estado procesada (P), suspendida (W) o bloqueda temporalmente (X).
Aviso. La tarjeta ya se encuentra activa o no se encuentra en un estado válido para realizar la activación.
Formulario
Realiza el bloqueo de la tarjeta. Para bloqueos temporales, la tarjeta debe estar activa (A). La cancelación de una tarjeta puede hacerse en cualquier momento y estado.
Formulario
Permite reasignar una nueva tarjeta a un cliente en remplazo de una tarjeta inactiva (estado C). La tarjeta a ser reasignada debe haber sido bloqueada previamente. La nueva tarjeta no debe estar activa, ni estar asociada a algún cliente. Esta operación no le altera su estado.
Aviso. La tarjeta a reasignar no se encuentra cancelada.
Formulario
Permite ordenar la emisión de una tarjeta virtual a tarjeta física. Esta orden solo será realizada una vez. Las tarjetas que ya hayan sido creadas como físicas no podrán hacer uso de este servicio.
Aviso. La tarjeta no es virtual.
Aviso. La emisión de la tarjeta virtual a física ya fue realizada.
Formulario
Una vez que el cliente prepago dispone de su tarjeta física, que ha nacido a partir de una tarjeta virtual, esta debe de habilitarse para las compras con lectura de tarjeta. Si no se invoca a este servicio no se podrán realizar compras con lectura de tarjeta desde ningún origen.
Aviso. La tarjeta nacio física o no se ha realizado la emisión de la tarjeta física.
